# 快速测试指南

## 测试资金流信号卡片

### 1. 启动 GUI

```bash
cd /Users/easonchen/WorkBuddy/Claw/output/alphalith/code
python3 -m alphalith.gui
```

### 2. 打开浏览器

访问 `http://localhost:8000`

### 3. 运行分析

1. 点击左侧「🔍 投研分析」
2. 输入标的代码（推荐用 **603986** 兆易创新，这只股票有龙虎榜数据）
3. 点击「▶ 开始分析」
4. 等待分析完成（约 30-60 秒）

### 4. 查看资金流信号卡片

分析完成后，在「II. Research Team Decision」和「III. Trading Team Plan」之间，会出现一个**「🔋 资金流信号」**卡片，包含：

- 🐉 **龙虎榜**：净买入金额、涨跌、上榜原因、买卖席位
- 💰 **大宗交易**：笔数、金额、折溢价率、买卖双方
- 🟢 **北向资金**：日级流向、近 3/5/10 日变动、个股持股比例
- 🔥 **市场热点**：板块热度归因
- 🔓 **解禁日历**：未来解禁日期、股份数量、占流通比

## 故障排查

### 卡片不显示
- 确认标的是 **A 股**（龙虎榜/大宗交易/北向资金只支持 A 股）
- 确认分析完成（进度条到 100%）
- 打开浏览器开发者工具（F12），查看 Console 是否有错误

### 信号数据为空
- 可能是当天没有龙虎榜数据（周末/节假日）
- 可能是网络问题（东财 API 偶尔超时）
- 查看终端日志，看是否有错误
